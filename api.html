book

Representa um livro.

GET /api/book/

Recebe uma lista de todos os livros cadastrados no formato JSON. Exemplo:

POST /api/books/

Adiciona um novo livro na biblioteca. O corpo da mensagem deve estar em formato JSON. Os seguintes campos são necessários:

  • title: Título do livro.
  • authors: Autores do livro (separados por vírgula).
  • desc: Descrição do livro.
  • price: Preço do livro.
  • photo: Foto da capa do livro.

PUT /api/books/:id

Substitui um livro já existente. Um documento no formato JSON contendo os mesmos atributos do método POST deve ser enviado no corpo da mensagem.

DELETE /api/books/:id

Deleta um livro existente com id infomrado.
